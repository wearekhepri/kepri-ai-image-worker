# Inputs

Place tes images de test ici.

Le script `kie_client_with_image.py` va :
1. Scanner ce dossier pour trouver les images
2. Les uploader vers l'API Kie.ai
3. Les utiliser comme input pour la génération

## Restrictions Nano Banana Pro

**Formats acceptés :**
- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)

**Limites :**
- Maximum 8 images par génération (officiellement jusqu'à 10-14 mais 8 recommandé)
- Taille max par image : 30 MB
- Peut maintenir l'identité de jusqu'à 5 personnes à travers les images

**Exemples d'images à tester :**
- Photo d'un vêtement à plat
- Photo d'un mannequin avec des vêtements
- Photo de produit à transformer en photo studio
- Plusieurs angles d'un même produit pour composition
